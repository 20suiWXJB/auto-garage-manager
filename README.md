## 平台简介

本项目是一个汽车修理厂管理系统，基于若依（RuoYi）前后端分离版本进行二次开发，面向中小型修车厂的日常经营与业务管理场景。

系统在保留若依成熟权限体系、后台管理能力和前后端分离架构的基础上，结合修理厂业务特点，可用于客户档案、车辆信息、维修工单、配件库存、结算对账、员工权限等业务模块的扩展与落地。

适用对象：

* 中小型汽车维修厂
* 汽车快修快保门店
* 需要数字化管理维修流程、客户和库存的汽修经营团队

技术基础：

* 前端采用Vue、Element UI。
* 后端采用Spring Boot、Spring Security、Redis & Jwt。
* 权限认证使用Jwt，支持多终端认证系统。
* 支持加载动态权限菜单，多方式轻松权限控制。
* 高效率开发，使用代码生成器可以一键生成前后端代码。

## 项目特点

* 基于若依成熟框架进行二次开发，便于快速扩展汽修行业功能。
* 适合中小修车厂使用，兼顾部署成本、使用门槛和后续维护。
* 采用前后端分离架构，便于后续对接小程序、H5 或第三方系统。
* 可结合门店实际业务继续扩展维修项目、配件管理、会员管理和财务统计等功能。
